Better Auth Ruby tracks upstream Better Auth `v1.6.9`. The Ruby port focuses on
server-side behavior: Rack apps, framework adapters, database adapters, auth
routes, cookies, sessions, and server plugins.

<Callout type="info">
Browser clients, TypeScript inference, React hooks, and native client storage are
outside the Ruby server scope unless a Ruby package explicitly documents an
equivalent.
</Callout>

  const tone =
    status === "Supported"

    <div className={`rounded-lg border p-4 ${tone}`}>
