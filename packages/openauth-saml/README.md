# openauth-saml

Alias package for `better_auth-saml`.

## Installation

```ruby
gem "openauth-saml"
```

```ruby
require "openauth/saml"
```

This package depends on `better_auth-saml` and loads the canonical Better Auth Ruby SAML 2.0 service provider helpers.

For documentation, visit https://better-auth-rb.vercel.app/.
