# openauth-oidc

Alias package for `better_auth-oidc`.

## Installation

```ruby
gem "openauth-oidc"
```

```ruby
require "openauth/oidc"
```

This package depends on `better_auth-oidc` and loads the canonical Better Auth Ruby enterprise OIDC relying-party helpers.

For documentation, visit https://better-auth-rb.vercel.app/.
